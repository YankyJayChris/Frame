Frame Framework Features

1. Variables

Define global variables with :vars.

:vars {
    $primary: "#FF0000";
    $spacing: "10dp";
}

2. Internationalization (i18n)

Support translations with :i18n.

:i18n {
    title: "Hello World";
    submit: "Submit";
}

3. Imports

Import files or specific components.

import "components.fr"
import { Card as BlogCard } "./components/Card.fr"

4. Pages

Define routes with page:.

page: {
    name: "Home"
    route: "/"
    styles: { background: "#FFFFFF" }
    children: [
        Text: { content: "Welcome" }
    ]
}

5. Components

Custom (component) and native components.

component Card: {
    props: { title: string = "Untitled" }
    styles: { width: "200dp" }
    content: "$title"
}

Text: { content: "Native Text" }

6. Props

Pass parameters to components.

component Button: {
    props: { label: string = "Click Me" }
    content: "$label"
}

7. Styling

CSS-like styles with variable substitution.

Text: {
    styles: { color: "$primary"; margin: "$spacing" }
    content: "Styled"
}

8. Content

Text content with i18n support.

Text: {
    content: t:"title"
}

9. Children

Nested components.

Container: {
    children: [
        Text: { content: "Child 1" },
        Button: { content: "Click" }
    ]
}

10. Animations

Define transitions.

Button: {
    content: "Fade In"
    animate: {
        from: { opacity: "0" }
        to: { opacity: "1" }
        duration: 500ms
        easing: "ease-in"
    }
}

11. Events

Handle user actions and lifecycle events.

input: {
    value: "Enter text"
    on_change: "update:($value)"
}

12. Functions

Custom logic with expressions.

fn update:(value:string) => {
    if: value === "test" {
        return result = "matched"
    }
}

13. State Management

Implicit via functions and props.

fn set_text:(text:string) => {
    return self.text = text
}

14. Navigation

Route navigation in functions.

fn go_home:() => {
    navigate("/home")
}

15. Fetch

HTTP requests.

fn load_data:() => {
    fetch:("https://api.example.com", "method:GET")
}

16. Forms

Form handling with validation.

form: {
    validation: "required"
    on_submit: "submit:()"
    children: [
        input: { value: "Name" }
    ]
}

17. Lists

Dynamic list rendering.

list: {
    data: posts
    build:(post) => {
        Card: { content: "$post.title" }
    }
}

18. Testing

Jest-like tests.

describe: "Button Tests" => {
    it: "click works" => {
        expect: button.on_click:("handle_click")
    }
}

19. Hot Reload

CLI feature: frame preview.

20. Deployment

CLI feature: frame deploy [target].

21. Camera Plugin

Access the device camera (Android, iOS, Desktop).

fn open_camera:() => {
    return result = camera:()
}

Button: {
    content: "Open Camera"
    on_click: "open_camera:()"
}

22. Location Plugin

Get device location (Android, iOS, Desktop).

fn get_location:() => {
    return loc = location:()
}

Button: {
    content: "Get Location"
    on_click: "get_location:()"
}

23. Notification Plugin

Send notifications (Android, iOS, Desktop).

fn send_notification:(msg:string) => {
    return result = notification:(msg)
}

Button: {
    content: "Notify"
    on_click: "send_notification:(\"New Post!\")"
}
